# 雙北 YouBike 場站配置模擬器

互動式網頁工具，協助分析雙北 YouBike 場站的車輛配置策略。

## 功能特色

- **3269 個雙北場站**完整資料（5天平均借減還，保守取整）
- 5 個篩選器：場站搜尋、城市、行政區、責任區、分級
- 三種解的分級：最優解、次優解、最後解
- 即時互動：可調整任一時段的「調入/調出車輛數」，庫存軌跡與圖表立即連動
- 🪄 **自動求最少調度**：演算法自動找出達到最優解所需的最少調度次數與時刻
- 圖表視覺化：每小時在站車輛數曲線，紅虛線=車位上限、橘線=見底

## 在 GitHub Pages 部署

1. 將整個 `youbike-simulator` 資料夾上傳到你的 GitHub repo
2. 進入 repo 的 Settings → Pages
3. Source 選 `Deploy from a branch`，分支選 `main`，資料夾選 `/youbike-simulator`（或將檔案放到根目錄則選 `/(root)`）
4. 等待 1-2 分鐘部署完成，網址為 `https://<你的帳號>.github.io/<repo名稱>/`

## 檔案說明

- `index.html` — 主程式（單一檔案，所有資料已內嵌，無需後端）

## 計算邏輯

- 每個整點時刻（6:00 ~ 23:00）的庫存需 > 0 且 < 總車位數
- 6:00 配置（A）和 15:00 配置（B）為主要兩個配置點
- 任意其他時段可加調度，演算法用貪婪法找最少需要幾次調度
